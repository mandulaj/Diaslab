# Diaslab
A web enabled modular weather station

  <img height="257" src="https://raw.githubusercontent.com/zpiman/Diaslab/master/artwork/diaslogo.png">
</p>

### Features to be implemented
* Temperature
* Humidity
* Luminosity UV/IR/Visible
* Pressure
* Gas
  * CO - MQ-7
  * CO2
  * Air quality - MQ-135
  * Pollution - MQ-3
* Wind
  * Speed
  * Direction
* Precipitation
* Rain drops
* Soil moisture
* Optical Dust sensor
* Geiger counter
